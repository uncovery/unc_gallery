Instructions

In order to display gallery contents in a page, post or other element of your Wordpress site, please use the [unc_gallery] keyword. You can add several settings to the keyword. Without settings, it will simply display the latest date's photos.

First of all you have to decide what type of information you want to display. This is done with the type="something" instruction. There are 3 different types. Please note that only specifying the type is not enough to display something.

  • day: This will display all images of one day. So you would write [unc_gallery type="day"]
  • image: This will display one single image only. So you would write [unc_gallery type="image"]
  • icon: This will display only the icon of a single image. So you would write [unc_gallery type="icon"]

Once you specified the type, you need to specify which day's photos you want to show. For this, you use the "date" instruction. You can either specify a specific date, the latest or a random one. If you want to display a specific date, you need to provide the date in the format "YYYY-MM-DD". So for example, for the 31st of January 2016, you write date="2016-01-31". If instead you want the latest or a random date, you just use those words instead: date="latest" or date="random". So with the type, the whole instruction would look like one of these:

  • [unc_gallery type="day" date="latest"]or
  • [unc_gallery type="image" date="random"]or
  • [unc_gallery type="icon" date="2016-08-14"]

For the image and icon options, the system will by default display the latest image (or icon) of the day chosen with the date setting. In order to display another image of the day, you need to use the file="filename.jpg" setting. So with a more specific setting, the previous list might look like this:

  • [unc_gallery type="day" date="latest"]or
  • [unc_gallery type="image" date="random" filename="image2.jpg"]or
  • [unc_gallery type="icon" date="2016-08-14" filename="image3.jpg"]

On top of this, there are also options for each type. You specify options with the options="something" setting. Each display type has different options:

  • day: datepicker. The datepicker option will show a monthly calendar on which all dates with photos are highlighted.
  • image / icon link. Normally, the image is just displayed by itself. With the "link" option, the image or icon will be linked to a page where the whole day can be seen as a day gallery.

With these options, the above examples might look like this:

  • [unc_gallery type="day" date="latest" options="datepicker"]or
  • [unc_gallery type="image" date="random" filename="image2.jpg" options="link"]or
  • [unc_gallery type="icon" date="2016-08-14" filename="image3.jpg" options="link"]
